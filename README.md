# 📂 Projects

## Phase 1 – Docker Fundamentals

### Project 01 – Dockerized Java Application
Containerize a Java application using Docker and understand image lifecycle management.

### Project 02 – Docker Compose Multi-Container Environment
Deploy multiple services using Docker Compose.

### Project 03 – Jenkins CI/CD Pipeline Automation
Implement automated CI/CD pipelines using Jenkins.

### Project 04 – GitHub Actions CI/CD Automation
Build cloud-native CI/CD workflows using GitHub Actions.

### Project 05 – Docker Registry Image Management
Manage container images using private and public registries.

---

## Phase 2 – Kubernetes & GitOps

### Project 06 – Kubernetes Application Deployment
Deploy containerized applications to Kubernetes clusters.

### Project 07 – Helm Kubernetes Package Management
Package and manage Kubernetes applications using Helm.

### Project 08 – ArgoCD GitOps Deployment Platform
Implement GitOps deployment strategies using ArgoCD.

### Project 09 – Prometheus Monitoring & Alerting Platform
Monitor applications and infrastructure using Prometheus.

### Project 10 – ELK Stack Log Management Platform
Centralized logging using Elasticsearch, Logstash, and Kibana.

---

## Phase 3 – Infrastructure Automation

### Project 11 – Terraform Infrastructure as Code Lab
Provision infrastructure using Terraform.

### Project 12 – Ansible Configuration Management Platform
Automate server configuration and administration using Ansible.

### Project 13 – AWS Cloud Infrastructure Deployment Platform
Deploy cloud infrastructure on Amazon Web Services.

### Project 14 – Azure Cloud Infrastructure Platform
Provision enterprise infrastructure on Microsoft Azure.

---

## Phase 4 – Advanced DevOps Engineering

### Project 15 – Advanced Jenkins Enterprise CI/CD Platform
Enterprise CI/CD implementation with quality and security controls.

### Project 16 – Kubernetes Security & Container Hardening Platform
Implement container security and Kubernetes hardening practices.

### Project 17 – Istio Service Mesh Platform
Manage secure microservice communication using Istio.

### Project 18 – OpenTelemetry Distributed Tracing Platform
Implement enterprise observability using OpenTelemetry and Jaeger.

### Project 19 – Advanced GitOps Platform with ArgoCD
Enterprise-grade GitOps deployment automation.

### Project 20 – Enterprise DevOps Platform
Complete end-to-end DevOps platform integrating all previous technologies.
