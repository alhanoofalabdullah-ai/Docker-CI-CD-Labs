# 🚀 Project 14 – Microsoft Azure Cloud Infrastructure Platform

## Overview

This project demonstrates cloud infrastructure deployment on Microsoft Azure using Terraform.

The project simulates enterprise cloud environments where infrastructure is provisioned, managed, and maintained using Infrastructure as Code (IaC).

Azure is one of the most widely adopted cloud platforms used by enterprises, governments, financial institutions, and technology organizations.

---

# Business Scenario

A company wants to deploy its applications into Microsoft Azure.

The cloud engineering team needs to provision:

- Resource Groups
- Virtual Networks
- Subnets
- Network Security Groups
- Virtual Machines
- Public IP Addresses

The deployment must be:

- Automated
- Secure
- Repeatable
- Scalable
- Governed
